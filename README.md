Chat App Project
Welcome to the Chat App Project! This web application facilitates communication through 1-to-1 and group chats, featuring different user roles with distinct permissions. Users can have roles such as Admin, Moderator (Comitateur), and Normal, allowing for effective control and moderation within the chat system.

Pages

Entry point for users to access the chat application.
Users can log in with their credentials to start chatting.

Permission Modifier Popup
Pop-up functionality for moderators (Comitateur) to modify user permissions.

Ban Popup (for Comitateur) and Logout (for Admin)
Popup for moderators (Comitateur) to ban specific users.

Admins have the ability to log out all users with the "bout" action.
Roles and Permissions
Admin: Full control, including user management and system actions.
Moderator (Comitateur): Moderation privileges, such as modifying permissions and banning users.
Normal: Standard user role with regular chat functionalities.
